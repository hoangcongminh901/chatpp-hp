About Chat++
 
Chat++ is an all-in-one Chatwork Enhancement Toolkit.
It is a free Chrome Extension which provides more funny emoticons as well as many other features for Chatwork.
Chat++ Features
 
■　Additional Emoticons for Chatwork.
■　Use your own Emoticons Data.
■　Easily Mention a person by typing @.
■　Mention all people in a box by typing @all. Mention your self by typing @me.
■　Mention people in your friend list even when they are not present in current chat box.
■　Mention a person or all people without name by adding _
■　Just display user picon (without mentioning) by adding .
■　Mention Group of people by typing only Group name.
■　Many useful shortcuts.
■　Easily check Room Information.
■　And more awesome features ...
Install Chat++ 3
 
 
You can download and install Chat++ via the Google Chrome Web Store!
 
Install from Store Download
If you can not install Chat++ from Chrome App Store, then you can try to install it from the source code.
Just click the DOWNLOAD button above, extract the zip file. After that, go to chrome://extensions/ page, turn on Developer Mode then click Load Unpacked Extension button.
Finally, refer to the folder named src, which is inside the folder that you just have extracted.
CREATE YOUR OWN EMOTICONS DATA
 
You can create your own Emoticons data file and then share it with your friends. Of course if you and your friends have the same data, you will see the same emoticons on Chatwork! You can easily create a Emoticons Data file for Chat++ by following the below rules:
  • The data file (as well as the change log file) must be written in json format.
  • The data file must have the following data:
    • data_name: Your data name (It should not be updated when you update your data file)
    • data_version: Your data version (It should be updated when you update your data file)
    • emoticons: Emoticons data. It is an array with each element is an emoticon's data.
    • You can refer the default emoticons data of Chat++ for more details.
  • You need to host your data file on the internet so Chat++ can access it! (https is required! So we suggest Dropbox!)
  • Finally, share the link to your data file with your friends and enjoy your own funny Emoticons!
